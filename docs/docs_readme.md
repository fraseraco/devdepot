docs – Project Documentation
This folder contains supporting documentation for the /dev/depot eCommerce project. These files provide insights into the system’s design, structure, and purpose, especially for development teams and academic presentations.

Included Files:
•	- ArchitectureDiagram.svg: High-level system architecture showing component interaction
•	- ER_Diagram.pdf: Entity-Relationship diagram of the database schema
•	- REST-API-ref.pdf: Reference for API endpoints, methods, and parameters
•	- Submission_SymposiumPoster_*.pdf: Visual poster used for project presentation
•	- docs_readme.md: Additional notes and summaries related to the documents

Usage:
Use these documents to understand project context, review API integration points, and communicate the system design to stakeholders or for academic submission.
